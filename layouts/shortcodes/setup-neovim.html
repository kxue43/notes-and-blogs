{{- /* setup-neovim.html */ -}} ## Set up neovim ```bash git clone https://github.com/kxue43/nvim-files ~/.config/nvim && nvim ``` After plugin installation finishes, run `:MasonInstallAll` to install all LSPs and `:TSInstallAll` to install all `tree-sitter` parsers. Run `:checkhealth` to see if there are any problems.
